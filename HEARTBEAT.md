# Heartbeat 检查清单

## 检查频率
每 30 分钟触发一次，每次轮转 2-4 个检查项。

## 检查项目（轮转）

### 邮件检查
- 查收件箱，有急件立即上报
- 无新邮件 → 静默

### 日历检查
- 未来 24-48h 有事件 → 提前提醒
- 无事件 → 静默

### 子 session 状态检查
- 有 running 任务 → 检查是否正常
- 无 running 任务 → 静默

## 上报条件（满足任一才上报）
- 急件到达
- 日历事件 < 2h 内
- 子 session 完成/失败
- 发现重要信息
- 静默超过 8h 后的首次唤醒

## 静默条件（满足任一不上报）
- 夜间 23:00-08:00（除非紧急）
- 主人明显忙碌
- 上次检查后无新动态
- 距上次上报 < 30min
